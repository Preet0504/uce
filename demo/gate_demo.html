The gate also enforces access, deterministically

None of the 24 real talkai responses happened to propose touching a restricted path — Claude avoided it correctly on its own every time. To confirm the deny branch actually fires (not just in unit tests), this is a constructed probe run live against the same real graph.

propose_change() verdict
BLOCK
Denied by rule 'DENY_NON_ADMIN_RBAC_WRITE' (requires role>=admin).
An editor may write freely under src/* — but src/rbac/* requires admin, regardless of what else is in the same change. This is evaluated the same way for every call; it is not a heuristic and does not depend on the model, the prompt, or how the request is phrased.
